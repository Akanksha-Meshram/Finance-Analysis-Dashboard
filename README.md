# Finance-Analysis-Dashboard
Interactive Power BI dashboard for transaction analytics, featuring dynamic KPIs, custom visualizations, and insights into gross amounts, taxes, wages, and verification status with a clean, professional layout.

**📌 Project Overview**

This dashboard provides a comprehensive view of financial performance by transforming raw transactional data into meaningful business insights. It enables users to monitor KPIs, analyze customer segments, evaluate transaction performance, and identify financial trends using interactive filters.
